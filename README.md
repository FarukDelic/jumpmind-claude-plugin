# JumpMind Claude Plugin

A Claude Code plugin designed for JumpMind development workflows, including commerce platform development, documentation, and implementation planning.

## Features

### Agents
- **doc-writer**: Specialized agent for creating and maintaining documentation
- **refactor-planner**: Agent for planning and executing code refactoring tasks

### Commands
- **commit**: Intelligent git commit message generation
- **implementation-plan**: Generate detailed implementation plans for features

### Skills
- **executing-plan**: Execute implementation plans step-by-step
- **playwright-skill**: Browser automation and testing capabilities

## Installation

Install this plugin using the Claude Code marketplace:

```bash
/plugin marketplace add https://github.com/FarukDelic/jumpmind-claude-plugin
/plugin install jumpmind-claude-plugin
```

Or install directly from the repository:

```bash
/plugin add https://github.com/FarukDelic/jumpmind-claude-plugin
```

## Usage

After installation, the plugin's commands, agents, and skills will be available in your Claude Code workspace.

### Commands
Use slash commands to access plugin functionality:
- `/commit` - Generate intelligent commit messages
- `/implementation-plan` - Create detailed implementation plans

### Agents
Agents will be auto-activated based on task context or can be manually triggered.

### Skills
Skills can be invoked using the skill system in Claude Code.
